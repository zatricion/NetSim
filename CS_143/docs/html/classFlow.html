void Flow::sendManyPackets ( double  time)

Send as many packets as possible.

Every packet that has never been marked as being sent (i.e. every packet that is in unsentPackets) that also satisfies windowStart <= sequence_num <= windowEnd will be sent. The ones that have already been sent (the ones not in unsentPackets) will not be resent.

Parameters:
timethe time at which the packets are sent.
