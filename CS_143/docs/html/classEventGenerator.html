Detailed Description

Superclass for Links, Hosts, Routers, and FlowGenerators.

EventGenerators each have their own eventHeaps, which are ordered lists of events. The EventHandler will determine the Event in the eventHeaps of the set of EventGenerators with the minimum time, and handle the Event.
